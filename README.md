# 🌌 Nebula AI Chatbot

A premium, full-stack AI chatbot application featuring a unique "Nebula" aesthetic and Mistral-7B intelligence.

## ✨ Features
- **Neural Link UI**: A unique, animated mesh-gradient interface.
- **Organic Design**: Tactile, irregular dialogue bubbles for a more human feel.
- **Mistral-7B**: High-performance AI processing via Hugging Face Inference API.
- **State Persistent**: Remembers your neural links (chat history) across sessions.
- **Dual Spectrum**: Seamless dark and light modes.

## 🚀 Deployment

### 1. Backend
```bash
cd backend
npm install
npm run dev
```

### 2. Frontend
```bash
cd frontend
npm install
npm run dev
```

## 🛠️ Stack
- **Frontend**: React (Vite), Framer Motion, Lucide Icons, Axios.
- **Backend**: Node.js, Express, Hugging Face API.
- **AI Model**: `mistralai/Mistral-7B-Instruct-v0.2`
